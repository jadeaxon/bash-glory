# bash-glory
My Bash library.

## Installation
For Cygwin:
```
./install.sh
```

Alternately, for Debian:
```
cd debian
make
dpkg -i *.deb
```

## Usage
```
source /usr/share/lib/bash-glory/<library>.sh
```
